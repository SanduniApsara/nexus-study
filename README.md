🎓 Nexus Study — Student GPA & Study Planner

> A full-stack MERN web application that helps university students manage modules, track GPA, plan study sessions, and analyze academic performance in real time.

📁 **Source Code:** https://github.com/SanduniApsara/nexus-study  

---

## ✨ Key Features

- 🔐 Secure Authentication — JWT-based login & registration  
- 📚 Module Management — Manage subjects with credits & grading  
- 📊 GPA Calculator — Real-time weighted GPA calculation  
- ✅ Task Manager — Track assignments with deadlines & priority  
- 📅 Study Planner — Log study sessions with productivity insights  
- 📈 Analytics Dashboard — Interactive charts (Bar, Line, Radar, Pie, Heatmap)  
- 🎯 Goal Tracking — Monitor target GPA progress  
- 🔔 Smart Alerts — Deadline reminders & overdue warnings  
- 📱 Responsive UI — Mobile-friendly modern interface  
- ☁️ Persistent Storage — MongoDB database  

---

## 🛠️ Tech Stack

### Frontend
- React 18 + Vite  
- Tailwind CSS  
- Zustand (state management)  
- Recharts (data visualization)  
- Axios  

### Backend
- Node.js  
- Express.js  
- MongoDB + Mongoose  
- JWT Authentication  

### Architecture
- RESTful API  
- Modular MVC structure  
- Client-server separation  

---

## 📁 Project Structure


nexus-study/

## 🔌 API Overview

- Modules (CRUD + grading)  
- Tasks (CRUD + filtering)
